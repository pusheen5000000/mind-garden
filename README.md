# 🌱 Mind Garden

Welcome to **Mind Garden**, a collection of interactive games designed to help users relax, reflect, and build skills for managing anxiety.

Mind Garden provides a calming space where users can explore different activities that encourage mindfulness, emotional awareness, and positive thinking. 🌿

## 🌸 About

Mind Garden transforms anxiety management strategies into simple, engaging games. Each activity focuses on different skills such as recognizing thought patterns, practicing relaxation, and checking in with emotions.

The goal is to make mental wellness tools approachable, interactive, and enjoyable for users of all ages.

## 🎮 Games

### 🧠 Thinking Errors
Learn to identify common thinking patterns and challenge unhelpful thoughts.

### 🌬️ Relaxation
Practice calming techniques through breathing and mindfulness activities.

### 🎣 Anxiety Review
Reflect on feelings and learn about coping strategies through an interactive fishing game.
